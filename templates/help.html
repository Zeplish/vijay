iIL13Pred Webserver


This tool helps the user to predict whether a given peptide can induce IL-13 or not.


Predict

This module allows the users to predict whether a given peptide can induce IL-13 or not.



Design

This module generates mutants of a given peptide and predicts their IL-13 inducing probability. This server helps users to find the minimum number of mutations needed to boost the IL-13 induction potential of query peptides.



Protein Scan

This module generates overlapping peptides from the query protein sequence and estimates their IL-13 induction activity. It helps the users to determine which segments of a protein have IL-13 induction ability.
